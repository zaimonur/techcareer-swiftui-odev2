# 📱 Techcareer SwiftUI for iOS Bootcamp — Ödev 2  
### SQLite Tabanlı Yapılacaklar (To-Do) Uygulaması  

SwiftUI kullanarak **SQLite** veritabanıyla CRUD (Create, Read, Update, Delete) işlemlerinin gerçekleştirildiği yapılacaklar listesi uygulaması.

---

## 🎯 Özellikler
- ✅ Yapılacakları **listeleme**
- ✅ Yapılacaklarda **arama**
- ✅ Yeni görev **kaydetme**
- ✅ Görevleri **güncelleme**
- ✅ Görevleri **silme**
- ✅ SQLite veritabanı ile **kalıcı veri saklama**
